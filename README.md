## 🎮 About

**Cosmic Rescue Squad** is a space-themed turn-based strategy game!

7 crew members (Astronaut, Medic, Pilot, Engineer, Scientist, Navigator, Captain) are all stacked at the Launch Pad. Beam them through 9 sectors to reach the Space Station — but the ORDER they arrive in earns you bonus points!

| One action per turn | ✅ Strictly enforced by UI |
| Forward only | ✅ Launch always goes +1 sector |
